# basic_copula_theory

このリポジトリは，江村 剛志 広島大教授 Ph.D. 著「シリーズ 情報科学における確率モデル 12 コピュラ理論の基礎」(コロナ社) (以下，「教科書」)に登場する図表をPythonで描画することで，コピュラ理論の理解を深めることを目的として作成された私的な学習用コード集です。
